# 12. 契約管理系画面仕様

## SCR-009 契約申込一覧

### 目的
契約申込を検索し、申込状況を確認する。

### 検索条件
- 申込番号
- 加入者番号
- 契約者氏名
- 被保険者氏名
- 商品コード
- ステータス
- 申請日From/To

### 一覧項目
- 申込番号
- 契約者氏名
- 被保険者氏名
- 商品名
- 保障プラン
- ステータス
- 申請日
- 更新日時

### 主な操作
- 詳細表示
- 新規申込作成

## SCR-010 契約申込詳細

### 目的
申込情報、審査結果、履歴を参照する。

### 主な表示セクション
- 基本情報
- 契約者情報
- 被保険者情報
- 商品・保障情報
- 告知事項
- 添付情報
- 審査履歴
- ステータス履歴

### 主な操作
- 編集（DRAFT/RETURNED 時のみ）
- 申請
- 再申請
- 審査詳細へ遷移

## SCR-011 新規契約申込作成

### 目的
新規契約申込を作成する。

### 入力項目
- 契約者加入者ID
- 被保険者加入者ID
- 商品コード
- 保障プラン
- 契約開始希望日
- 契約終了予定日
- 保険金額
- 支払方法（月払 / 年払）
- 告知事項（喫煙有無、既往歴有無、入院歴有無）
- 備考
- 身分確認書類添付有無

### 主な操作
- 一時保存
- 申請
- 取消

### バリデーション
- 契約者必須
- 被保険者必須
- 商品コード必須
- 契約開始希望日必須
- 契約終了予定日必須
- 契約開始希望日 <= 契約終了予定日
- 保険金額 > 0

### 業務ルール
- 契約者と被保険者は同一でも別でもよい
- 保障プランは商品ごとの候補から選択する

## SCR-012 契約一覧

### 目的
有効契約・解約済契約等を検索する。

### 検索条件
- 契約番号
- 契約者氏名
- 被保険者氏名
- 商品コード
- 契約ステータス
- 契約開始日From/To

### 一覧項目
- 契約番号
- 契約者氏名
- 被保険者氏名
- 商品名
- 契約開始日
- 契約終了予定日
- ステータス

### 主な操作
- 詳細表示

## SCR-013 契約詳細

### 目的
契約情報と関連手続きを参照し、各種手続きを起票する。

### 主な表示セクション
- 契約基本情報
- 契約者・被保険者情報
- 保障情報
- 変更履歴
- 給付請求履歴
- 添付情報

### 主な操作
- 契約変更申請作成
- 解約申請作成
- 給付請求作成

### 活性条件
- ACTIVE の契約のみ変更申請・解約申請・給付請求を起票可能

## SCR-014 契約変更申請作成

### 目的
契約変更を申請する。

### 入力項目
- 変更種別
- 変更適用希望日
- 変更理由
- 新住所関連項目（住所変更時）
- 新電話番号、新メールアドレス（連絡先変更時）
- 変更後保障プラン、変更後保険金額（保障内容変更時）

### 主な操作
- 一時保存
- 申請
- 取消

### 表示制御
- 変更種別に応じて入力セクションを切り替える

## SCR-015 解約申請作成

### 目的
契約の解約を申請する。

### 入力項目
- 解約希望日
- 解約理由
- 備考

### 主な操作
- 申請
- 取消

### バリデーション
- 解約希望日必須
- 解約理由必須
- 解約希望日は契約開始日以降
