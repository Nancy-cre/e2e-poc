# 15. 管理系画面仕様

## SCR-022 ユーザー一覧

### 目的
システム利用ユーザーを検索、参照する。

### 検索条件
- ログインID
- 氏名
- ロール
- 有効状態

### 一覧項目
- ユーザーID
- ログインID
- 氏名
- メールアドレス
- 主ロール
- 有効フラグ

### 主な操作
- 新規登録
- 編集

## SCR-023 ユーザー登録・編集

### 目的
ユーザー情報とロール割当を登録・編集する。

### 入力項目
- ログインID
- 氏名
- メールアドレス
- 仮パスワード
- 有効フラグ
- ロール複数選択

### 主な操作
- 保存
- 取消

### バリデーション
- ログインID必須
- 氏名必須
- メールアドレス必須
- 新規登録時は仮パスワード必須
- ロール1件以上必須

## SCR-024 ロール一覧

### 目的
ロール定義を参照する。

### 表示項目
- ロールコード
- ロール名
- 説明

### 備考
PoC では参照のみでも可。編集機能は任意。

## SCR-025 保険商品一覧・編集

### 目的
保険商品を管理する。

### 一覧項目
- 商品コード
- 商品名
- 商品種別
- 最低保険金額
- 最高保険金額
- 請求上限額
- 有効フラグ

### 入力項目
- 商品コード
- 商品名
- 商品種別
- 説明
- 最低保険金額
- 最高保険金額
- 請求上限額
- 有効フラグ

### 主な操作
- 新規登録
- 更新
- 停止

## SCR-026 監査ログ一覧

### 目的
主要操作の監査ログを検索、参照する。

### 検索条件
- イベント種別
- 実行ユーザー
- 対象種別
- 対象ID
- 実行日時From/To

### 一覧項目
- 監査ログID
- イベント種別
- 実行ユーザー
- 対象種別
- 対象ID
- 実行日時
- 結果

### 備考
詳細ポップアップまたは折りたたみで before/after を参照できるようにする
